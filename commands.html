General
/help
All
Opens the in-game help menu with a full command list.
/spawn
All
Teleport back to the main server spawn.
/rules
All
View the server rules — please read them!
/discord
All
Get the Discord invite link in chat.
/rankup
All
View rank upgrade options and pricing.
/report <player> <reason>
All
Report a player to the moderation team.
Homes
/sethome [name]
All
Set a home at your current location. Name is optional (defaults to "home").
/home [name]
All
Teleport to one of your set homes.
/delhome <name>
All
Delete a saved home by name.
/homes
All
List all your saved homes and their coordinates.
Teleport
/tpa <player>
All
Send a teleport request to another player.
/tpaccept
All
Accept an incoming teleport request.
/tpdeny
All
Deny an incoming teleport request.
/warp [name]
All
Teleport to a server warp. Run /warp without arguments to list all available warps.
/back
VIP+
Teleport to your last location before a teleport or death.
/rtp
All
Random teleport to a safe location in the wilderness.
Economy
/bal
All
Check your current in-game balance.
/baltop
All
View the richest players on the server.
/pay <player> <amount>
All
Send money to another player.
/jobs
All
Open the Jobs menu to earn money by performing in-game tasks.
/jobs join <job>
All
Join a specific job (e.g. Miner, Farmer, Hunter).
/ah
All
Open the auction house to buy and sell items with other players.
/ah sell <price>
All
List the item in your hand on the auction house at the given price.
Claims
/claim
All
Claim the chunk you are standing in. Hold a golden shovel and right-click corners for custom shapes.
/unclaim
All
Remove the claim you are currently standing in.
/claimlist
All
View all your active claims and remaining claim blocks.
/trust <player>
All
Grant a player full build & interact access to your claim.
/untrust <player>
All
Remove a player's trust from your claim.
/containertrust <player>
All
Allow a player to open containers (chests, furnaces) in your claim without full build access.
Shops
/shop
All
Open the server shop to buy basic resources.
/pshop
All
Open the player shop browser to buy from other players' shops.
/pshop create <price>
All
Create a player shop listing for the item in your hand.
/pshop remove
All
Remove one of your active shop listings.
Chat
/msg <player> <message>
All
Send a private message to another player.
/r <message>
All
Reply to the last private message you received.
/ignore <player>
All
Ignore a player's messages.
/nick <nickname>
Elite+
Set a custom display nickname. Supports colour codes.
/tag <tag>
God+
Set a custom chat prefix tag displayed before your name.
Rank Perks
/fly
Elite+
Toggle creative flight in your own claim or designated zones.
/gfly
God+
God-mode fly — faster flight speed with no stamina limit.
/feed
God+
Instantly refill your hunger bar. 5 minute cooldown.
/heal
MVP
Instantly restore full health. 10 minute cooldown.
/ptime <day|night>
VIP+
Change your personal client-side time of day (does not affect other players).
/pweather <sun|rain>
Elite+
Change your personal client-side weather.
/hat
VIP+
Wear any block as a helmet for cosmetic purposes.
